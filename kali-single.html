How to install Kali Linux in Oracle WM VirtualBox

  • Lucky Ngabuh
  • Web Design
  • 14
Create a new Virtual disk (VDI, dynamic allocation etc...)

Step 1 - Create a new Virtual Machine

Before proceeding, make sure you have successfully installed VirtualBox. On GNU/Linux you can check this by opening a terminal and typing virtualbox and/or the icon to start VirtualBox will be visible in the menu. For other operating systems there will be a similar icon to start it.

Step 1.1 - Enter a name for your Virtual Machine

Enter Kali Linux as the name or any name you like. Load Kali Linux ISO (click on the arrow at your right, and search for the ISO in the download folder.) and proceed, Type and Version will be detected automatically.

Step 1.2 - Allocate RAM and CPU

Allocate at least 2 GB of RAM for optimal performance and allocate not more than 25% of your system RAM for better practices. The more RAM you give your virtual machine, the better and faster it will be.
choose 1 GB of RAM

The calculations of the amount of memmory/RAM are as follows: 1.00 GB = 1024 MB
2.00 GB = 2048 MB
3.00 GB = 3072 MB
4.00 GB = 4096 MB
and so on.
If You want 6 GB, just multiply it with 1024. Your answer obtained is the amount of memory/RAM you want to allocate.

Step 1.3 - Create a Virtual Hard Drive

On the next screen select Create a Virtual Hard Disk Now. Set the disk size at 20 GB for Home Edition and 40 GB for Security Edition This will create a Dinamically Allocated Disk, if you want to have the entire disk size, check "Pre-Allocated Full Size"
